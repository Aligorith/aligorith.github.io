Blender Development Work

Since 2006, I've been an active core Blender developer, focussing primarily on the Animation System/Tools and Grease Pencil.

Animation System/Tools

Notable contributions:

  • Animato (2009) - Architect of a full recode of the animation system to make it possible to "animate everything", as part of the 2.5 project.
    • Animation Editors Design + Implementation (2008-2011) - Full redesign of the animation editors (and all their tools) to
    • AnimChannel Filtering API (2008-2011) - Architect of the underlying backbone used to power all the animation editors.
    • Keying Sets (2009)
  • Constraint System Refactor (2007) - Refactored the constraints system to be more modular, more easily extensible, and more flexible. Developed the following constraints:
    • Transform Limiting Constraints
    • Limit Distance
    • Clamp To
    • Damped Track
    • Transformation
    • PyConstraint
  • New Dependency Graph Engine Architecture(DEG) (2013)
  • Bullet RigidBody Sim Integration (2010)
  • Bendy Bones (2016)
  • Spline IK
  • Breakdowner, Push/Pull Pose
  • Pose Propagate
  • Pose Libraries
  • Bone Colour Sets + Bone Groups
  • Mask Modifier
